# New repo workflow from local files

## Add the remote repository to the local directory

```bash
git init
```

```bash
git add -A
```

```bash
git commit -m "Initial commit"
```

```bash
git branch -M main
```

```bash
git remote add origin {REPO SSL URL}
```

```bash
git push -u origin main
```

## List existing remotes

```bash
git remote -v
```
