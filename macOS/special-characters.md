# Special Characters

| Description       | Symbol  |
| :---------------- | :------ |
| Command (or Cmd)  | ⌘       |
| Shift             | ⇧       |
| Option (or Alt)   | ⌥       |
| Control (or Ctrl) | ⌃       |
| Caps Lock         | ⇪       |
| Arrows            | ↑ ↓ ← → |
| Enter             | ⮐      |
| Apostrophe double, open (“) | ⌥  [ |
| Apostrophe double, close (”) | ⌥ ⇧  [ |
| Apostrophe single, open (‘) | ⌥  ] |
| Apostrophe single, close (’) | ⌥ ⇧  ] |
